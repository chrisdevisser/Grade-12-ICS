Assignment

Complete the following assignment and submit your work to the dropbox.

This assignment requires that you complete three programs.

Program 1 – Fibonacci Numbers

Save the following program as Fibonacci. Zip up the folder to hand in.

Use a recursive function to find the value of the nth Fibonacci number. Recall that the Fibonacci numbers are listed as follows: 1, 1, 2, 3, 5, 8, 13, 21, 34 . . .

Where:

F(1) = 1

F(2) = 1

F(n) = F(n-1) + F(n-2) for all n>2

Sample Application Output

Program 2 – n Choose r

Save the following program as NChooseR. Zip up the folder to hand in.

First, write a recursive function to calculate n! (n factorial).

Use this n! function to calculate the number of ways to choose r different objects from a set of n objects.

NOTE: Feel free to use the code from the content for this.

Sample Application Output

Program 3 – Fraction Reducer

Save the following program as ReduceFraction. Zip up the folder to hand in.

First, write a recursive function to calculate the GCD of two integers.

Use this GCD function to reduce a fraction that is input by the user.

NOTE: Feel free to use the code from the content for this.
